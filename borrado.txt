Si necesitas limpiar datos (solo para ti como administrador):
Pega esto temporalmente en la consola cuando lo necesites:

javascript
// Solo para uso administrativo - NO incluir en el código público
if (confirm('¿BORRAR TODOS los datos? Esto es IRREVERSIBLE.')) {
    localStorage.removeItem('todasLasSolicitudes');
    sessionStorage.removeItem('solicitudes_respaldo');
    // Limpiar backups
    Object.keys(localStorage).forEach(key => {
        if (key.startsWith('backup_')) localStorage.removeItem(key);
    });
    alert('Todos los datos han sido eliminados. Recargando...');
    location.reload();
}

-------------------------------------
Si necesitas hacer cambios futuros:
Para limpiar datos (solo para administración), ejecuta en la consola:

javascript
// Solo para uso administrativo
localStorage.clear();
sessionStorage.clear();
location.reload();
